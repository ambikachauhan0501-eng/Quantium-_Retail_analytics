# Quantium-_Retail_analytics
This project is part of the Quantium Retail Analytics internship task.   The objective is to analyze transaction and customer datasets to uncover purchasing behaviors, identify sales drivers, and provide actionable recommendations for the chip category strategy.
# Retail Analytics Project – Customer Segmentation & Sales Insights

## Tasks & Workflow

### 1. Transaction Data Examination
- Check for inconsistencies, missing values, and outliers.
- Verify category items are correctly identified.
- Ensure numeric data is consistent across all tables.
- Apply necessary cleaning steps and save the cleaned dataset.

### 2. Customer Data Examination
- Inspect customer dataset for nulls and anomalies.
- Clean and standardize fields.
- Merge transaction and customer datasets once validated.
- Save intermediate files for reproducibility.

### 3. Data Analysis & Metrics
- Define key metrics:
  - Total sales
  - Drivers of sales
  - Highest sales by region, product, or customer segment
- Perform exploratory analysis with charts and graphs.
- Document interesting trends and insights.

### 4. Deep Dive into Customer Segments
- Segment customers based on purchasing behavior.
- Evaluate packet size relevance and preferences.
- Provide recommendations on which segments to target.
- Form overall conclusions based on analysis.

### 5. Deliverables
- Cleaned datasets saved as CSV files.
- Visualizations (bar charts, line plots, heatmaps).
- Analytical report with insights and recommendations.

##  Learning Objectives
- Gain hands‑on experience in **data cleaning** and **customer analytics**.
- Develop skills in **merging datasets** and ensuring data integrity.
- Learn to define and calculate **business metrics**.
- Practice **visual storytelling** through charts and graphs.
- Provide **strategic recommendations** based on data insights.

##  Tools & Libraries
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook  

## How to Run
1. Clone the repository.  
2. Open the Jupyter Notebook 
3. Run cells sequentially to reproduce cleaning, analysis, and visualizations.  
4. Review CSV outputs and charts for insights.  

## Expected Outcomes
- Clean, reliable transaction and customer datasets.  
- Clear understanding of sales drivers and customer purchasing behavior.  
- Segmentation strategy highlighting target groups.  
- Actionable recommendations for chip category planning.
